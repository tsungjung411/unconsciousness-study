## 人多必有白痴＆天才
- 記憶培訓，就如同補習班騙局，人多必然會有人考上台大
  - 你也要看看母體數有多大，如果有 1000 人來補習，1人考上台大，那也只是個案。台大上榜率 = 0.1％
  - 但補習班會拿此誇大，來我這邊補習的人能考上台大，荒謬！
  - **樹多必有枯枝，人多必有白痴!樹不要皮，必死無疑；人不要臉，天下無敵。**
  - 如果有 1000 人來補習，200 位都考上台大，那才叫做厲害。台大上榜率 = 20％
  - 記憶培訓班的狀況，就如同補習班騙局一樣。
